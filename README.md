# SurviveXJTU

西安交通大学生存指南。

## 本地部署与调试

安装 [Node.js](https://nodejs.org/zh-cn)/[Git](https://git-scm.com/)。

```shell
git clone https://github.com/SurviveXJTU/SurviveXJTU.github.io.git
cd SurviveXJTU.github.io
npm install -g pnpm
pnpm install
pnpm run docs:dev
```
